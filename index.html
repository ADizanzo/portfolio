Educación

Tecnicatura en Desarrollo Web y Aplicaciones Digitales

Instituto Superior Politécnico de Córdoba 2022 - 2024

En esta carrera nos capacitamos para crear y mantener sitios web y aplicaciones. Aprendimos sobre programación, desarrollo de interfaces, gestión de bases de datos, y diseño de soluciones digitales. Durante la carrera, adquirimos conocimientos sobre lenguajes de programación (Python, Javascript y Java), frameworks, y metodologías ágiles. El plan de estudios también incluye el análisis de necesidades de los usuarios y la optimización de experiencias digitales, brindando herramientas para diseñar productos de calidad adaptados a las demandas del mercado.

Data Analytics

Codo a Codo 2023

Aprendimos análisis de datos utilizando herramientas y técnicas actuales. Trabajamos con grandes volúmenes de información, interpretamos datos y generamos informes claros y comprensibles. Profundizamos en herramientas como Excel, Power BI, y lenguajes de programación como Python. Este curso nos preparó para aplicar métodos estadísticos y algoritmos de machine learning para extraer insights y tomar decisiones informadas basadas en datos.

Javascript & NodeJs

Codo a Codo 2023

Desarrollamos aplicaciones web dinámicas utilizando dos de las tecnologías más populares. El curso cubre desde los fundamentos de JavaScript hasta el desarrollo de servidores y APIs con Node.js. Aprendimos a crear interfaces interactivas y a gestionar bases de datos en el back-end. Además, abordamos prácticas como la gestión de dependencias y el uso de frameworks y librerías, lo que nos permite dominar el desarrollo full-stack con un enfoque práctico y actual.

Curso de Java

Educación IT 2022

En este curso aprendimos a desarrollar aplicaciones utilizando uno de los lenguajes de programación más populares. Durante el curso, profundizamos en conceptos fundamentales como la orientación a objetos, estructuras de datos, y manejo de excepciones. Además, aprendimos a crear aplicaciones robustas y eficientes, optimizando el código y utilizando herramientas de desarrollo modernas. Este curso nos prepara para enfrentar desafíos técnicos en el desarrollo de software en diversos entornos.

Full Stack

Alura Latam 2022

Formación completa en desarrollo web, orientado tanto al front-end como al back-end. Trabajamos con diversas tecnologías y lenguajes de programación, como HTML, CSS, JavaScript, Node.js y bases de datos. A través de una metodología práctica y proyectos reales, adquirimos las competencias necesarias para desarrollar aplicaciones web completas, desde la creación de interfaces interactivas hasta la implementación de la lógica de servidor y la gestión de bases de datos.
